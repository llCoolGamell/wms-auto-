# WMS Automator v3.1

Автоматизация терминала WMS: подстановка кодов в поля «Контроль» и нажатие Ок / F2 по циклу складских операций.

## Исправление v3.1

На экране **«Поиск паллеты»** в «Контроль» подставляется код **только из поля «Паллета»**, а не из «Из места» (типичный сбой старых версий).

## Требования

- Windows
- Python 3.10+ (для сборки из исходников)
- Запуск **от имени администратора** (нужно для `keyboard` и доступа к окнам WMS)

## Быстрый старт

1. Положите рядом с программой Excel со зонами (колонка A — название зоны, B — код ячейки), например `zones.xlsx`.
2. Запустите `dist\WMS_Automator.exe` **от администратора** или: `python wms_auto.py`
3. Укажите файл зон (или положите `zones.xlsx` в папку с exe).
4. Fallback по умолчанию: `D-KM-1`.
5. Нажмите **«Начать перемещение»**, откройте WMS на любом шаге цикла.
6. **Любая клавиша** — пауза.

## Сборка exe

```bat
build.bat
```

Результат: `dist\WMS_Automator.exe`

## Цикл WMS

| Окно | Действие |
|------|----------|
| Главное меню | F2 — запросить работу |
| Взятие работы | F2 — взять работу |
| Перемещение к источнику | Место → Контроль |
| Поиск паллеты | **Паллета** → Контроль |
| Поиск коробки | Коробка → Контроль |
| Поиск места-приёмника (коробка) | Код из Excel по зоне |
| Размещение в место | Ок |
| Поиск места-приёмника (паллета) | Fallback-ячейка |

## Файлы

- `wms_auto.py` — основной код и интерфейс (тёмная тема v3)
- `wms_auto.log` — лог рядом с exe
- `zones.xlsx` — ваш справочник зон (не в git, добавьте свой)
